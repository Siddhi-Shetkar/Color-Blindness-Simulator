# Color Blindness Simulator

## Overview
A web application that allows users to upload an image and preview how it appears under different types of color vision deficiencies. Users can compare the original image with the simulated version and switch between different simulation modes.

## Features
- Upload an image
- Preview the original and simulated images
- Simulate Protanopia, Deuteranopia, Tritanopia, and Achromatopsia
- Compare original and simulated images
- Download the simulated image

## Tech Stack
- HTML
- CSS
- JavaScript
- Canvas API
- EXIF.js
